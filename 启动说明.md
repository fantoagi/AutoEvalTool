# 工具启动说明

## 方法1：开发环境启动（推荐用于开发和测试）

### 步骤1：打开命令行
- **Windows**: 按 `Win + R`，输入 `cmd` 或 `powershell`，按回车
- 或者在项目文件夹中，按住 `Shift` 右键，选择"在此处打开 PowerShell 窗口"

### 步骤2：切换到项目目录
```powershell
cd "d:\个人\cf\cf\ZY\项目\招商证券2025\10-需求沟通跟踪\智能体\CompareTools\DemoV3"
```

### 步骤3：启动应用

**方式A：使用 streamlit 命令（推荐）**
```powershell
streamlit run app.py
```

**方式B：如果 streamlit 命令不可用，使用 Python 模块方式**
```powershell
python -m streamlit run app.py
```

**方式C：如果上述方式都不行，使用完整路径**
```powershell
python run_gui.py
```

### 步骤4：访问应用
启动成功后，浏览器会自动打开，或者手动访问：
- http://localhost:8501
- http://127.0.0.1:8501

## 常见问题

### Q1: 提示 "streamlit 不是内部或外部命令"
**解决方案**：
```powershell
# 使用 Python 模块方式
python -m streamlit run app.py
```

### Q2: 端口 8501 被占用
**解决方案**：
1. 查看占用端口的程序：
   ```powershell
   netstat -ano | findstr :8501
   ```
2. 结束占用端口的进程，或让 Streamlit 自动使用下一个可用端口（8502、8503等）

### Q3: 无法访问页面
**检查**：
1. 查看控制台输出，确认服务器是否成功启动
2. 确认防火墙没有阻止连接
3. 尝试访问 http://127.0.0.1:8501 而不是 http://localhost:8501

### Q4: 打包后的 exe 无法启动
**解决方案**：
1. 查看控制台错误信息
2. 确认所有依赖都已正确打包
3. 检查 `config` 和 `辅助小结` 目录是否在 exe 同目录下

## 快速启动脚本

### Windows 批处理文件（开发环境）
创建 `start.bat` 文件：
```batch
@echo off
cd /d "%~dp0"
python -m streamlit run app.py
pause
```

### Windows 批处理文件（打包版本）
创建 `start_exe.bat` 文件：
```batch
@echo off
cd /d "%~dp0\dist"
start AutoEvalTool.exe
```

双击这些 `.bat` 文件即可快速启动。
