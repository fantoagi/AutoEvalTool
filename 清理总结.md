## 注意事项

1. **不要删除的文件：**
   - 所有 `.py` 核心代码文件
   - `build.py` 和 `AutoEvalTool.spec`（打包必需）
   - `打包经验总结.md` 和 `打包快速参考.md`（重要经验）
   - `config/` 和 `辅助小结/` 目录

2. **可以手动清理的：**
   - `__pycache__/` - Python 缓存（可删除，会自动重建）
   - `build/` - 构建临时文件（可删除，打包时会重建）
   - `.DS_Store` - macOS 系统文件（可删除）

3. **日志文件：**
   - `process.log` 和 `summary.log` 是运行时生成的，可以保留用于问题排查
