# Streamlit 打包快速参考卡

## 🚀 快速打包（3步）

```powershell
# 1. 激活虚拟环境
.\venv\Scripts\Activate.ps1

# 2. 运行打包脚本
.\打包应用.bat

# 3. 测试
.\dist\AutoEvalTool.exe
```

## ⚠️ 最常见错误及一键修复

### 错误1：RuntimeError: server.port does not work...

**原因：** 开发模式未禁用

**修复：** 在 `run_gui.py` 中添加：
```python
os.environ['STREAMLIT_GLOBAL_DEVELOPMENT_MODE'] = 'false'
config_content = """[global]
developmentMode = false
"""
```

### 错误2：Streamlit 导入卡住

**修复：**
```batch
强制清理.bat
# 然后重启计算机
```

### 错误3：404 NOT FOUND

**修复：** 先修复错误1，然后重新打包

### 错误4：Object.hasOwn is not a function

**原因：** 浏览器版本过旧

**修复：** 
1. 升级浏览器到 Chrome 93+, Edge 93+, Firefox 92+
2. 或使用其他现代浏览器
3. 查看 `浏览器兼容性说明.md` 获取详细说明
